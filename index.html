Level 1, Module 1: Cryptography Foundations & Lab Setup

Lab Log Cryptography

1. Sandbox Environment & Hygiene

Deployed a Type-2 hypervisor to host an isolated Kali Linux virtual machine. Configured a NAT network adapter to maintain internet connectivity for dependency installation while restricting inbound lateral traffic to the host operating system.

Hardened default machine configurations by immediately rotating default root credentials via passwd and synchronizing rolling-release package distributions using sudo apt full-upgrade.

2. Symmetric Cryptography (AES-256)

Utilized OpenSSL to perform symmetric data-at-rest encryption. Evaluated the role of key stretching with -pbkdf2 and 100,000 hashing iterations (-iter 100000) to defend against hardware-accelerated brute-force calculations.

Troubleshooting Checkpoint: Diagnosed and fixed a bad magic number error caused by an inverted input flag. Analyzed how OpenSSL evaluates critical file signatures (checking for header bytes like Salted__) to classify encrypted vs. cleartext file structures.

3. Asymmetric Cryptography (GPG & ECC)

Generated asymmetric key pairs via GNU Privacy Guard (GPG), utilizing modern Elliptic Curve Cryptography via the cv25519 curve. Successfully orchestrated public-key distribution simulations to enforce strict recipient-only data decryption boundaries.

4. Integrity & Digital Signatures

Executed SHA-256 calculations to establish cryptographic digital fingerprints. Demonstrated the Avalanche Effect by shifting a single character string and observing a completely non-linear transformation of the output hash string.

Wrapped up the session by deploying clear-signed documents (--clearsign) to ensure data authenticity and cryptographically enforce non-repudiation metrics within communication channels.
